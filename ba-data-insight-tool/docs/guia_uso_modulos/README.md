# Guia de uso por modulo

Esta carpeta documenta como usar cada espacio de trabajo de la plataforma APP BA.

Archivos incluidos:

- `01_pagina_principal.md`: selector de espacios de trabajo.
- `02_ba_insight.md`: uso del flujo BA Insight.
- `03_treasury_tower.md`: uso del modulo Treasury Tower.
- `04_reconciliation_e2e.md`: estado y alcance del modulo ReconciliationE2E.
- `05_model_of_costs.md`: uso del modulo Model of Costs.
- `Guia_Uso_Modulos_APP_BA.xlsx`: workbook con una hoja por modulo.

Los modulos marcados como MVP en desarrollo pueden tener pantallas preparadas sin logica productiva completa.
